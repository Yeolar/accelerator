accelerator
===========

accelerator是一个C++基础库，有一些组件修改自Facebook的folly基础库。采用C++11语法，主要关注于基础组件的高性能实现。

依赖包括 Boost DoubleConversion GFlags ICU OpenSSL ZLIB。

编译安装

    $ mkdir build && cd build
    $ make -j8
    $ sudo make install
